## คำอธิบายโปรเจกต์ (Project Description)

โปรเจกต์นี้คือการพัฒนาระบบเว็บไซต์ที่ช่วยให้ผู้ใช้งานสามารถ **วางแผนและจัดทำรายการอาหารประจำสัปดาห์** (Weekly Menu Planning) โดยมีการ **แนะนำเมนูอาหารเพื่อสุขภาพ** (Healthy Menu Recommendation)

เว็บไซต์นี้ถูกพัฒนาขึ้นเพื่อเป็นส่วนหนึ่งของรายวิชา **408-47-10 Computer Science Project 2  ** ประจำภาคเรียน 1 / 2568

## เทคโนโลยีและเครื่องมือที่ใช้ (Technology Stack)

รายการเครื่องมือและเทคโนโลยีหลักที่ใช้ในการพัฒนาโปรเจกต์นี้:

| ประเภท | เทคโนโลยี | รายละเอียด |
| :--- | :--- | :--- |
| **Backend** | PHP (Core) | ใช้เป็นภาษาหลักในการประมวลผลฝั่งเซิร์ฟเวอร์ จัดการตรรกะของระบบ และเชื่อมต่อฐานข้อมูล |
| **Frontend** | HTML / CSS / JavaScript | ใช้สร้างโครงสร้าง (HTML), จัดรูปแบบ (CSS), และสร้างการโต้ตอบแบบ Real-time (JS) |
| **Frontend Framework** | Bootstrap | ใช้เป็น UI Framework หลักในการจัดวาง Layout และออกแบบหน้าเว็บให้สวยงามและรองรับมือถือ (Responsive)) |
| **Database** | **MariaDB (Relational)** | ใช้เป็นระบบฐานข้อมูล (ที่มาพร้อม XAMPP) สำหรับจัดเก็บข้อมูลผู้ใช้ สูตรอาหาร และแผนอาหารทั้งหมด |
| **Web Server** | Apache | ใช้เป็น Web Server สำหรับรันโปรเจกต์ในสภาพแวดล้อมจำลอง (ผ่านโปรแกรม XAMPP) |
| **Version Control** | Git & GitHub | สำหรับการจัดการเวอร์ชันของโค้ดและการทำงานร่วมกันเป็นทีม |
| **Tools** | Composer, PHPMailer | ใช้ Composer จัดการ Library ภายนอก เช่น PHPMailer สำหรับการส่งอีเมล (รีเซ็ตรหัสผ่าน) |
